# 3.0 学员 Skill 合集 — 总目录

> 精选 55 个高频实用 Skill，覆盖 AI 增强、内容创作、开发工具、浏览器自动化、社交媒体、知识学习、效率工具、数据分析 8 大类别。

---

## 如何使用

1. **已包含技能**：将对应 `SKILL.md` 放入 `~/.claude/skills/技能名/` 文件夹，重启 Claude Code 即可触发
2. **SkillHub 技能**（标注 🌐）：需运行 `npx clawhub@latest install 技能名` 安装
3. 所有技能均通过 `/技能名` 或自然语言描述触发
